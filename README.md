## 📊 Research Project: AI-powered SEO Content Production

### Objective
To analyze how leading SEO practitioners use AI to scale content production, improve rankings, and optimize workflows.

### What I Did
- Identified high-quality SEO experts actively using AI
- Collected and structured insights from YouTube and content sources
- Transformed raw transcripts into actionable frameworks
- Focused on real-world execution, not theory

### Repository Structure
- /research/sources.md → List of experts and selection rationale  
- /research/youtube-transcripts/ → Structured insights from videos  

### Key Learnings
- AI SEO is shifting from backlinks → brand mentions  
- Retrieval sources are critical for AI search visibility  
- Human + AI hybrid workflows perform best  
- Content systems matter more than individual tools  

### Outcome
Built a structured research base that can be used to develop a real AI-powered SEO playbook.
